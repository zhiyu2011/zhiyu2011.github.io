学习使用Hexo搭建博客

Posted on 2022-11-24，4 min read

#准备工作：安装相关工具
1、安装Node.js
https://nodejs.org/dist/v4.2.3/node-v4.2.3-x86.msi
https://nodejs.org/dist/v4.2.3/node-v4.2.3-x64.msi
选择需要的版本下载，分别是32位和64位。下载后安装一直默认下一步就行。
2、安装Git
Git-2.11.0.3-32-bit.exe
Git-2.11.0.3-64-bit.exe
打开cmd命令行(win+r 输入cmd回车)，分别执行
node -v
npm -v
git --version

会得到相对应的版本信息

如果都可以成功运行出现版本信息证明安装成功。

#一、安装Hexo
必须按照步骤来，因为hexo需要使用node.js的npm命令。
打开cmd命令行(win+r 输入cmd回车)或者打开Git Bash，输入以下命令安装Hexo

npm install -g hexo-cli
或
npm install hexo-cli -g

等待自动安装完成后。输入
hexo -v
检查是否安装成功
到这一步如果所有都安装成功，并测试无问题，那么本地配置已经搞定，下面开始在github上建项目了。

hexo安装错误的坑，提示code FETCH_ERROR

安装时报错，网上找了好多办法试了都不行，最后换淘宝源安装就好了
第一种：修改权限，chmod 777 /var/root 修改权限
第二种：npm install -g -unsafe-perm hexo
第三种：使用淘宝镜像，npm install -g hexo --registry=https://registry.npm.taobao.org
第四种：分别执行以下命令
npm cache clean -f
npm install -g n
npm stable
npm -g install hexo-cli
还有一种（我没试）：http://stackoverflow.com/questions/15633029/npm-no-longer-working

#二、初始化hexo基础配置文件
创建一个文件夹（我是在E盘新建了个hexotext文件夹）用于存放hexo。然后右键该文件夹使用

git bash here
命令就可以直接定位到当前目录。
接着输入
hexo init
或者
打开Git Bash，输入命令 hexo init [folder]例 $ hexo init hexotext
[folder]代表的是你的hexo文件夹名，如果不写，就默认在当前文件夹初始化，会在当前目录下创建同名文件夹。然后进入你的hexo文件夹$ cd [folder]

接着输入npm install ——————————————————在这一步怎么都走不通，报错

配置node。然后输入

hexo g

加载hexo基础html、css、js等文件。
在这完成后等于已经在本地创建了一个网页，想查看的话，输入

hexo s

换电脑怎么办
很简单，直接将整个blog文件夹拷贝到那台电脑上就可以，前提是目标电脑也有node、git环境。

之后就都可以使用了，无论是本地的hexo操作还是提交到github上，都是可以操作了。
